## How to Add These Posts

1. Log in as admin at `/api/login`
2. Navigate to the blog admin interface
3. For each post, copy the metadata and content from the corresponding .md file
4. Create the post with the proper settings (published, category, tags, etc.)

All posts are SEO-optimized with:
- Descriptive meta descriptions

- Internal linking opportunities
- Brand voice consistency ("Enchanted Grit")
